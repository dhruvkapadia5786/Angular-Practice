Announcing Angular.dev

New documentation and a better UX for all levels of developer Angular.dev teaches modern Angular development from the start.

Some highlights include:

📕 A new content structure, focused on teaching core Angular concepts for beginners, and digging deeper into in-depth guides for advanced developers

🧱 All docs and examples are built with Standalone APIs for simplified development, including a fully rewritten Components guide

⚡️ Core web vitals guidance written in collaboration with Chrome’s Aurora team, including guides on Image Optimization and Performance Best Practices

🌊 New guides for SSR and Hydration, for better app performance by default

🔭 More helpful API Reference & CLI references (under development)

⚒️ Quick edit/refresh using the Vite dev server in our tutorials

🚗 An updated public Roadmap

🔍 Better search

+ a whole lot more!

Our team remains committed to stability and backward compatibility, cornerstones of the framework’s reliability. We have updated guides on Keeping up-to-date and NgModules to ensure Angular.dev is helpful for all teams.
